# Shardus Init

# Creates a new shardus project

# Synopsis

shardus init <appName> [templateRepo] [--no-install]

# Examples

Create a new Shardus app project from an existing template:

shardus init ./my-shardus-app https://gitlab.com/shardus/enterprise/applications/coin-app-template.git

# Description

shardus init <appName> [templateRepo] is used to create new Shardus app projects from existing template repositories.

A templateRepo is a git repository that has a _templates/shardus/new folder that contains hygen templates.

If the --no-install option is provided, npm install will not be run in the new project folder.
