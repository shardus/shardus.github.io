# Shardus Network

# Manage a local Shardus test network

# Synopsis

shardus network create [--default]
shardus network start
shardus network scale
shardus network stop
shardus network clean

# Examples

Create a local test network with default parameters:

cd ./my-shardus-app
shardus network create --default

Enter the created instances directory and start the test network:

cd instances/
shardus network start

Add more nodes to the network

shardus network scale

Stop the test network:

shardus network stop

Clean residual log and database files from the last run:

shardus network clean

# Description

The shardus network tools are used to launch and scale local test networks that run your Shardus app code.

shardus network create provides a questionnaire that gathers information about the way your app is configured. This includes the location of the main executable and configuration files for your application. It then creates a directory for the test network and saves the information it gathered into a network-config.json file there. The --default option can be passed to skip the questionnaire and use default values.
