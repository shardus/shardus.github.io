To verify the license file was notarized:

  1. Click the version number to download the file.
  2. Run the command: sha256sum [filename] to get the checksum of the file.
    Or upload the file here
  3. Click the 'Notary' link to open the transaction and then the 'Click to see more' link to get the 'Input Data' field.
  4. Ensure that the checksum matches the 'Input Data' field.
  5. If so, the license file was notarized at the Timestamp of the transaction.

To notarize a license file:

  1. Click the version number to download the file.
  2. Run the command: sha256sum [filename] to get the checksum of the file.
    Or upload the file here
  3. Go to MyEtherWallet
  4. Connect your MetaMask wallet or some other wallet. Make sure the wallet address is associated with your organization. Use a network with low fees; like Polygon.
  5. Send a transaction to your own address with zero native coins of the network. Be sure to set the 'Data' field under 'Advanced' to the checksum of the file starting with 0x.
  6. This transaction can be used as proof that you notarized the file at the Timestamp of the transaction.
